# УкрАвтоІмпорт — Партнерська програма (B2B / B2D)

Лендинг-сторінка для дилерів, автопарків та лізингових компаній: прямі поставки електромобілів і гібридів з Китаю в Україну.

## Структура

- `index.html` — розмітка сторінки (українською)
- `styles.css` — стилі (без зовнішніх фреймворків)
- `script.js` — мобільне меню, smooth-scroll, reveal-анімації, обробка форм

## Запуск локально

Сторінка статична — достатньо відкрити `index.html` у браузері або підняти простий http-сервер:

```bash
python3 -m http.server 8080
# далі: http://localhost:8080
```

## Секції лендинга

1. Hero з формою швидкого запиту та метриками
2. Переваги для дилерів (9 карток)
3. Формати співпраці: B2D, B2B-автопарки, лізинг, Trade-in fleet, реселлери, EV-charging
4. Покроковий процес (6 кроків)
5. Модельний ряд (BYD, Zeekr, Li Auto, Voyah, Xpeng, Geely)
6. Порівняння «з нами / самостійний імпорт»
7. Відгуки партнерів
8. FAQ
9. Контактна форма для дилерів
10. Footer з контактами та посиланнями

## Інтеграції

Форми наразі працюють у демо-режимі (без бекенду). Для бойового використання потрібно під’єднати:

- Endpoint для приймання заявок (`fetch`-запит у `script.js → форма submit`)
- CRM / Telegram-бот / email-розсилку для менеджерів
- Аналітику (GA4, Pixel) — додати у `<head>` `index.html`
