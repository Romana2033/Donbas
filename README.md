# УкрАвтоІмпорт — Партнерська програма (B2B / B2D)

Лендинг-сторінка для дилерів, автопарків та лізингових компаній. Інтегрується у наш сайт на WordPress (тема **Vehica** → дочірня тема **vehica-child**).

## Склад пакета

| Файл | Призначення | Куди ставити |
| --- | --- | --- |
| `page-dealers.php` | Page Template — розмітка сторінки на українській | `wp-content/themes/vehica-child/page-dealers.php` |
| `assets/css/dealers.css` | Стилі лендингу (бренд УАІ) | `wp-content/themes/vehica-child/assets/css/dealers.css` |
| `assets/js/dealers.js` | JS: smooth-scroll, reveal-анімації, AJAX-відправка форм | `wp-content/themes/vehica-child/assets/js/dealers.js` |
| `functions.snippet.php` | Snippet для `functions.php`: enqueue, CPT `uai_lead`, AJAX-обробник | додати у `functions.php` дочірньої теми |

## Встановлення

1. Скопіювати `page-dealers.php` у каталог дочірньої теми.
2. Скопіювати теку `assets/` у дочірню тему.
3. Вставити вміст `functions.snippet.php` у `functions.php` дочірньої теми.
4. У WP-адмінці: **Сторінки → Додати нову**, назва «Дилерам», у блоці **Атрибути сторінки → Шаблон** обрати «Партнерська програма (B2B / B2D)», опублікувати.
5. URL сторінки за замовчуванням — `/dealeram/` (або будь-який зручний slug).
6. У меню сайту додати посилання на сторінку.

## Інтеграції

- **Email-сповіщення**: за замовчуванням лист іде на `admin_email`. Змінити можна через фільтр:
  ```php
  add_filter( 'uai_dealer_lead_recipient', function () {
      return 'b2b@ukravtoimport.ua';
  } );
  ```
- **Telegram / CRM**: підключаються через action-хук `uai_dealer_lead_saved( $post_id, $fields )` — приклад у коментарі всередині `functions.snippet.php`.
- **Аналітика**: GA4/Pixel вже стоять глобально через Vehica. Для трекінгу подій надсилання форми додайте у `dealers.js` `gtag('event', 'dealer_lead', { source: ... })` в callback `onDone(true)`.
